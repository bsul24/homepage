# Homepage

A responsive personal homepage built for The Odin Project's Advanced HTML and CSS course. The page includes a hero/about section, a project gallery, and a contact footer, with layouts adapted for desktop, tablet, and mobile screens.

## Live Demo

[View Live Project](https://homepage-bsul24.netlify.app/)

## Features

- Responsive desktop, tablet, and mobile layouts
- Hero section with layered image/card styling
- Diagonal header background using CSS `clip-path`
- Project card grid with hover shadows
- Contact section with social and contact links
- Custom CSS variables for fonts and colors

## Built With

- HTML
- CSS
- CSS Grid
- Flexbox
- Media queries
- Google Fonts

## What I Practiced

- Building responsive layouts across multiple breakpoints
- Using semantic page sections
- Creating layered visual designs with CSS
- Styling reusable project cards
- Managing spacing, shadows, and image sizing
- Using CSS custom properties for theme values
